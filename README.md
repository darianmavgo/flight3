# Project Flight3

This project is organized into the following directory structure:

- **app_settings**: Stores application-level configuration files.
- **bin**: Contains compiled executable binaries.
- **cmd**: Holds the main application entry points.
- **internal**: Contains private application logic and internal packages.
- **logs**: Stores application log files generated during runtime.
- **sample_data**: Holds sample datasets used for development and testing.
- **static**: Contains static web assets such as CSS, JavaScript, and images.
- **templates**: Stores HTML templates for server-side rendering.
- **test_output**: Contains artifacts and logs generated from test executions.
- **tests**: Holds the project's test suites and integration tests.
- **user_settings**: Stores user-specific configuration and preferences.
